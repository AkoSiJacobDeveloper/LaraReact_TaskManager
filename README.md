A simple task management application built with Laravel, Inertia.js, React, and TypeScript. It allows users to create, update, complete, and track tasks with due dates and status filtering.

Features
- Create, edit, and delete tasks
- View task details
- Update task status
- Filter tasks by status
- Show overdue tasks based on due date

Tech stack
- Laravel
- Inertia.js
- React
- TypeScript
- MySQL
- Tailwind CSS
